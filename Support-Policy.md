# Support Policy

## Service

Cert4Sign is a hosted web service at [https://cert4sign.com](https://cert4sign.com) for generating ZATCA-compliant Certificate Signing Requests and completing e-invoicing compliance onboarding.

## Support Channels

- **Email**: support@cert4sign.com
- **GitHub Issues**: [https://github.com/hesham-fouda/cert4sign/issues](https://github.com/hesham-fouda/cert4sign/issues)
- **Wiki**: [https://github.com/hesham-fouda/cert4sign/wiki](https://github.com/hesham-fouda/cert4sign/wiki)

## What We Support

- CSR generation for all ZATCA environments (Sandbox, Simulation, Production)
- ZATCA compliance certificate issuance
- Compliance invoice testing (6 test invoice types)
- Production CSID acquisition
- Report generation and download

## What We Don't Support

- ZATCA portal account setup (contact ZATCA directly)
- OTP generation (must be obtained from [ZATCA Fatoora portal](https://fatoora.zatca.gov.sa))
- Custom invoice XML generation for production use
- Tax consultation or accounting advice
- Integration with third-party ERP systems

## Reporting Security Issues

For security vulnerabilities, please email **security@cert4sign.com** directly. Do not create public issues for security concerns.
